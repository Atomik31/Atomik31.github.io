Portfolio de **Julien Charlier** — Architecte Data / IA · IoT (LoRaWAN) & Modélisation prédictive · Double compétence OT-IT.

🔗 **Site en ligne : [atomik31.github.io](https://atomik31.github.io/)**

## Aperçu

- Parcours, compétences et projets : MLOps, IoT, computer vision
- Bilingue français / anglais
- Site statique en HTML / CSS / JavaScript vanilla, sans framework ni dépendance

## Structure

```
├── index.html            # Point d'entrée
├── styles.css            # Styles
├── app.js                # Rendu du contenu et bascule FR/EN
├── data/portfolio.json   # Tout le contenu du site (bilingue)
└── assets/               # Images et CV
```

Tout le contenu est centralisé dans `data/portfolio.json` : modifier ce fichier suffit à mettre le site à jour.
